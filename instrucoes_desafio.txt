

Criar funções para as operações existentes:
sacar, depositar e visualizar extrato

criar duas novas funções:
criar usuario (cliente do banco)
criar conta (vincular ao usuario)

Saque 
deve receber argumentos apenas por nome (keyword only)

sugestão input: saldo, valor, extrato, limite, numero_saques, limite_saques
sugestão output: saldo e extrato


deposito
apenas por posiçao (positional only)
sugestão args: saldo, valor, extrato
sugestão output: saldo e extrato

extrato
deve receber args por posicão e nome
args posicionais: saldo
args nomeados: extrato

____
Criar usuario (CLIENTE)
Deve armazenar usuarios em uma lista
usuario é composto por: 
nome
data nascimento
cpf
endereço (string com o formato: logradouro, nro - bairro - cidade/sigla estado)

Deve ser armazenado apenas os numeros do cpf
Não pode cadastrar 2 usuarios com mesmo CPF

____
Criar conta (vinculada a um cliente)
deve armazenar contas em uma lista
uma conta é composta por:
agencia
numero
usuario

o numero da conta é sequencial, iniciando em 1
numero da agencia é fixo "0001"
usuario pode ter mais de uma conta, 
mas uma conta pertence a somente um usuario


____
Dica
para vincular um usuario a uma conta, filtre a lista de usuarios
buscando o numero do CPF informado para cada usuario da lista
